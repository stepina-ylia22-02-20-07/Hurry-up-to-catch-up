Название игры:
    “Hurry up to catch up”
Проект написан на Python с использованием библиотеки PyGame
Идея:
    Создать игру лабиринт на Python с помощью библиотеки PyGame. 
    Пользователь играет за зайца, который убегает от лисы. 
    Суть игры - дойти до финиша, чтобы тебя не успели поймать.
Задачи:
    Придумать идею;
    Найти нужную информацию;
    Подобрать необходимые материалы;
    Реализовать проект;

Для создания карты использовалось приложение Tiled
